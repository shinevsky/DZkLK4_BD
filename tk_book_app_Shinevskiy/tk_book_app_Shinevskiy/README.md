# Tkinter Book Viewer (Chapter 2)

Стек:
- Python 3.9+,
- Tkinter (входит в stdlib),
- Pillow (PIL) для работы с изображениями

Что делает:
- Отображает обложку книги, название и автора.
- Кнопка для загрузки собственной фотографии обложки.

Установка:
1. python -m venv venv
2. source venv/bin/activate  # macOS/Linux
   venv\Scripts\activate     # Windows
3. pip install -r requirements.txt

Запуск:
$ python main.py

Примечание:
Tkinter используется для простых утилит; Pillow нужен для масштабирования изображений.
